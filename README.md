# Calculadora em C#

> Calculadora de simples execução em C#, ao qual deverá atender todos os requisitos básicos de cálculo.

### Requisitos

O projeto visava, em sua execução:

- [x] Reconhecer operadores;
- [x] Realizar cálculos de soma, subtração, divisão, multiplicação e porcentagem;
- [x] Validação de número, com limitação a zero.
